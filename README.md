## Сборка / Запуск


1. Установка зависимостей

```bash
npm install
```

2. Запуск проекта

```bash
npm run dev
# или используйте yarn
yarn dev
```

3. Сборка проекта

```bash
npm run build
# или используйте yarn
yarn build
```
